Noodle Shop Advert

Noodle Shop Advert

As we are building a modern website, we should also include some advertising. This way we can make loads of money and retire (in our dreams).

Add a simple advert for the Noodle restaurant

Create a div that uses a single row.

Inside the row, add a column of size 10 to take up most of the width of the page.

Use a column offset of 1 to centre this column in the middle of the page

<div class="row"> <!-- Noodle Soup Shop Advertisement
    <div class="col-sm-10 col-sm-offset-1">
      <div class="alert alert-info">
        <a class="pull-left"
           href="http://kungfupanda.wikia.com/wiki/Noodle_shop"
           target="_blank"><img id="mr-pings-noodle-shop-img"
                                src="images/mr-pings-noodle-shop.png"></a>
        <h4>Dragon Warrior Noodles & Tofu</h4>
        <p>
          The finest noodles and tofu in the whole of the Valley of Peace.
          Come and savour food as you have never tasted before,
          thanks to a secret ingredient only known by the Ping family.
        </p>
      </div>
    </div>
    </div>

Download the Mr Ping's noodle shop image and put it in your images directory.

Fixing the Image

Our image is far too big, so our advert takes up most of the page. As we have given the image in the advert an id tag, we can define styles and image size in our css/styles.css file.

Edit the css/styles.css file and add the following styles to make our image a suitable size for our advert

Lets set the height and width of the noodle shop image to 64px, that is four times as big as the default size of the text. We will also ad a margin to the right of the image to keep a nice gap between the image and the text.

#mr-pings-noodle-shop-img{
    height:64px;
    width:64px;
    margin-right:15px;
}
