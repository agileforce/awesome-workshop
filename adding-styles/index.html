Adding styles

Adding Basic Styles

So far our simple web page is lacking any real kind of awesomeness...

We can start to fix this by adding styles to our content. Styles can include different fonts, size and colour of text.

Replace your simple website with a basic website for Poh, using some basic styles to the web pate. Then reload the page in your browser to see the changes

<!DOCTYPE html>
<html>

  <head>
    <!-- Files containing styles will be added here later -->
  </head>

  <body style="background-color:powderblue;">

    <h1 style="color:blue;text-align:center;">Simple web page</h1>

    <p>
      Hello, my name is <strong>Poh</strong> and I am a Panda who loves Kung Fu.
    </p>

    <img src="images/kungfu-poh.gif"
         style="float:right;" />

    <h2>Career</h2>

    <p>
      I work for a family run noodle soup business that I run with my dad.  We provide nutritious soup and dough balls for the community we live in.
    </p>

    <h2>Hobbies</h2>
    <p>
      I love to spend my time learning Kung Fu.  I also collect memorabilia on the
      <a href="http://kungfupanda.wikia.com/wiki/Furious_Five"
         target="_blank">Furious Five</a>
    </p>

    <h2>Where to find me</h2>
    <p>
      I currently live in the
      <a href="http://kungfupanda.wikia.com/wiki/Valley_of_Peace">Valley of Peace</a>
    </p>

    <!-- The image tag is self-contained, it uses a closing / rather than a closing tag -->

    <img src="images/valley-of-peace.jpg"
         alt="The Valley of Peace"
         style="width:600px;height=480px;" />

  </body>
</html>

These are just a few examples of styles, some of which make the design better, some styles make the design look worse. You can spend a lot of time getting the styles right for your website.

Luckily there are lots of libraries of styles, like Bootstrap, that you can use to make your website look awesome with a lot less effort.
