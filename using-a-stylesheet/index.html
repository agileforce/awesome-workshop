Using a Stylesheet

Using a Stylesheet

Instead of putting styles directly into the HTML file, we can simplify things by putting the styles into their own file. This file is called a Stylesheet.

By having styles in one stylesheet, hey can be used for all your web pages.

Linking to a Stylesheet

To include the styles from a stylesheet in a web page you need to use the link tag.

Create a new stylesheet file for your styles an then add that stylesheet to your html file

With your favourite editor, create a file called styles.css inside a new sub-directory called css

Add the following styles to the styles.css file.

/* Simple style definitions */


/* Define a style for the body tag */
body{
    background-color:powderblue;
}


/* Define a style for all h1 tags */
h1{
    text-align: center;
    color:blue;
}


/* Define a style for a specific identity,
in this case an image */

#kungfu-poh-img{
    height:100px;
    width:100px;
    float:right;
}


/* Define a style for a specific identity,
in this case an image */

#valley-of-peace-img{
    width:600px;
    height:480px;
}

Link the new stylesheet

Add the stylesheet to the index.html page by including the following line in between the head tags

<link rel="stylesheet" href="css/styles.css">

Removing the inline styles

You can remove the inline styles from the index.html file now, so that file should look like this:

<!DOCTYPE html>
<html>

  <head>
    <!-- Files containing styles -->
    <link rel="stylesheet" href="css/styles.css">
  </head>

  <body>

    <h1>Simple web page</h1>

    <p>
      Hello, my name is <strong>Poh</strong> and I am a Panda who loves Kung Fu.
    </p>

    <img src="images/kungfu-poh.gif"
         alt="Poh practising his Kung Fu moves - animated gif"
         id="kungfu-poh-img" />

    <h2>Career</h2>

    <p>
      I work for a family run noodle soup business that I run with my dad.
      We provide nutritious soup and dough balls for the community we live in.
    </p>

    <h2>Hobbies</h2>
    <p>
      I love to spend my time learning Kung Fu.  I also collect memorabilia on the
      <a href="http://kungfupanda.wikia.com/wiki/Furious_Five"
         target="_blank">Furious Five</a>
    </p>

    <h2>Where to find me</h2>
    <p>
      I currently live in the
      <a href="http://kungfupanda.wikia.com/wiki/Valley_of_Peace">Valley of Peace</a>
    </p>

    <!-- The image tag is self-contained, it uses a closing / rather than a closing tag -->

    <img src="images/valley-of-peace.jpg"
         alt="The Valley of Peace"
         id="valley-of-peace-img" />

  </body>
</html>

Now, reload the index.html file in your browser and you should see the same
