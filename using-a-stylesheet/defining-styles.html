Defining styles

Defining Styles in a Stylesheet

You can define styles for you html file using several different methods

  • Using the html tag name
  • Using a unique Id
  • Using a Class name

Using an html tag name

You can specify one or more styles for a specific html tag, such as h1, body or p. This will apply the styles to all those tags throughout the web page.

This means you can easily add a style to make all your headings look the same, eg all have the same font, all be bold, all be the same size, etc.

In our example we defined a background colour for our website using the body tag and made all our h1 headings be centred and shown in the colour blue

We can do the same in our style sheet:

/* Define a style for the body tag */
body{
    background-color:powderblue;
}


/* Define a style for all h1 tags */
h1{
    text-align: center;
    color:blue;
}

As we are using the html tags, we do not need to change anything in our html file.

If you want to apply the same styles to multiple html tags, you can simply put them in a comma separated list:

/* Define a style for the body tag */
body{
    background-color:powderblue;
}


/* Define a style for all h1 tags */
h1,h2,h3{
    text-align: center;
    color:blue;
}

Using a unique Id

We can define a style or set of styles to apply to anything by using a unique name called an Id.

We use a # character followed by a unique name to define the Id. Then in curly brackets, { } we add the styles we want to apply to the content with this Id.

Here is an example of three styles we are applying to anything with the Id kungfu-poh-img

#kungfu-poh-img{
    height:100px;
    width:100px;
    float:right;
}

To use the styles for this Id in our html page, find the relevant tag and at the id to it.

So in this case as our style is for a particular image, we find the img tag in our html file and update it to include an id

<img src="images/valley-of-peace.jpg"
     alt="The Valley of Peace"
     id="valley-of-peace-img" />

Classes of styles

If you have a group of things you want to apply styles to, you can assign them all a class name. So if you wanted to apply styles to a group of images, rather than give each image a unique id, you can use a class.

You can of course define styles for the img tag, however this would apply to all img tags. A class can be used to apply styles to a specific set of img tags.

In the following example, all these images can have the same styles applied to them by using the img.friends name.

<div class="col-sm-3"><!-- Friends Oogway -->
         <p><img src="images/friends-oogway.png"
                 alt="Friends Oogway"
                 class="friends"/></p>
       </div>

       <div class="col-sm-3"><!-- Friends Shifu -->
         <p><img src="images/friends-shifu.jpg"
                 alt="Friends Shifu"
                 class="friends"/></p>
       </div>

       <div class="col-sm-3"><!-- Friends Tigress -->
         <p><img src="images/friends-tigress.jpg"
                 alt="Friends Tigress"
                 class="friends"/></p>
       </div>

So in the CSS file we can define styles for this img.friends class as follows:

img.friends{
    height:96px
    width:96px
}
