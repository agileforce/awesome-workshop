Create a simple web page

Lets dive in and create the simplest possible webs page. We will then cover what each of the pieces of the web pages does

Create a new file called index.html in your favourite editor and add the following code

<html>

  <head>
    <!-- Files containing styles will be added here later -->
  </head>

  <body>

    <h1>Simple web page</h1>

    <p>
      This is a little bit of content inside a paragraph tag.
    </p>

    <p>
      We can include links to other websites, 
      like <a href="https://google.co.uk">Google</a>
    </p>

    <!-- The image tag is self-contained, 
         it uses a closing / rather than a closing tag -->

    <img src="images/kungfu-poh.gif~c200" />

  </body>
</html>

The parts that make up this simple web page are explained in this section.
